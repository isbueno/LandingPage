Este é um projeto de desenvolvimento web feito no 3º Semestre do curso de ADS.

O objetivo deste site era reunir fãs de uma série de livros em uma comunidade no site.

O exercício proposto de fazer a manutenção foi resolvido nas páginas Home.html e Style.css, fazendo uma atualização de alguns termos de web semântica e de css em algumas partes do site.

O restante dos arquivos é irrelevante para o exercício.
